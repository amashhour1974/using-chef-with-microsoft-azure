Tools referenced in the book built by Pendrica

  • chef-provisioning-azurerm

    Chef Provisioning driver for Azure Resource Manager

  • kitchen-azurerm

    A driver that integrates the popular Test Kitchen tool with Azure Resource Manager

  • azure-credentials

    A tool that makes it easier for people to add Service Principals to Azure (nb: supports Azure Active Directory users only)

  • Chef Extension for Visual Studio Code

    Rich language support for Chef DSL and snippets when using Visual Studio Code, including Rubocop and Foodcritic linting support
